# Roadmap

- **More visualizers** — waterfall spectrogram, scrolling oscilloscope, configurable colour palettes.
- **Cross-platform media keys** — wire up the OS-level transport controls (play/pause/next on keyboards and headsets) on Windows, macOS, and Linux.
- **Configurable FFT** — runtime selection of FFT size, window function, and bin grouping; right now those are baked at compile time.
- **Optional remote control** — a tiny HTTP or named-pipe listener so a second tool can drive playback (handy for streaming setups and "now playing" widgets).

PRs welcome — see the GitHub issues tab for current threads.
