Conplaya is a modern .NET 9 console audio player that keeps everything inside a single terminal window: 24-bit Unicode album art, a live FFT-based graphic equalizer, a timing HUD, and keyboard-driven playback (seek, pause, playlist navigation).

Drop in any `.mp3`, `.wav`, `.aac`, `.aiff`, `.flac`, `.wma`, or `.m4a` file and Conplaya queues every supported track in the same folder. Press `v` at any time to swap between four full-width visualizers — a doom-style fire effect, classic bar EQ, mirrored waveform, and a 24-bit pixel pulse field — all rendered with `▀` half-block characters and ANSI true-colour pairs.

Released as a self-contained single-file `play.exe` via GitHub Actions + GitVersion + Chocolatey, with no external .NET runtime required.
